Object Tagging Instructions

You will see 5 images of rotating objects that allow you to see the objects in different views.

Please give tags for the objects in the aspects of categories, colors, shapes, and materials. If you have multiple tags for one aspect, please separate the tags with commas and more tags are encouraged (means you can get a bonus :)). Note that the tags must meet the following criteria:

  1. Each tag must be a single word
  2. Each tag cannot be longer than 25 characters
  3. Each tag must match the corresponding aspect

Example:


chair
square
brown, silver
wooden, steel
